# רצף - מעקב הרגלים

אפליקציית PWA בעברית למעקב אחר הרגלים יומיים, מותאמת למסך iPhone.

## מה יש באפליקציה

- מסך "היום" עם אחוז התקדמות יומי
- הוספה, עריכה ומחיקה של הרגלים
- סימון הרגלים כבוצעו
- בחירת יום קודם וסימון ✓ או X על תאריך שכבר עבר
- מעקב רצפים ותובנות שבועיות
- לוח חודשי עם ימי התקדמות
- שמירה מקומית בדפדפן באמצעות `localStorage`
- סנכרון לענן דרך GitHub Gist פרטי
- סטטיסטיקה נפרדת לכל הרגל
- יעדי רצף של 7, 30, 60 ו־100 ימים
- סימון יום כבוצע או כ־X
- תמיכה ב־RTL ועברית
- מניפסט ואייקונים להתקנה למסך הבית

## הפעלה

אפשר לפתוח את `index.html` ישירות בדפדפן, או להגיש את התיקייה עם שרת סטטי.

```bash
python -m http.server 4173
```

ואז לפתוח:

```text
http://localhost:4173/
```

## סנכרון ענן

במסך "תובנות" אפשר לחבר GitHub token עם הרשאת `gist`.
האפליקציה יוצרת Gist פרטי בשם `retzef-habit-data.json` ושומרת אליו את נתוני ההרגלים.

ה־token נשמר רק בדפדפן של המכשיר שבו מזינים אותו.
